# @ube-tsp/ky-emitter

TypeSpec emitter generating namespace-structured TypeScript using Alloy-JS.

Based on [typespec-zod](https://github.com/bterlson/typespec-zod) implementation patterns.

## Usage

```yaml
# tspconfig.yaml
emit:
  - "@ube-tsp/ky-emitter"
```

```bash
npm install @ube-tsp/ky-emitter
tsp compile .
```
